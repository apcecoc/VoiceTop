## üìñ Description (English):  
The **VoiceTop** module analyzes voice messages in a chat and generates a leaderboard of users who have sent the most. Perfect for monitoring user activity or creating fun statistics for group chats.  

### üõ† Commands:  
- `.voice [number_of_messages]` ‚Äî Counts voice messages and shows the top users.  
  - **Examples:**  
    - `.voice` ‚Äî Analyzes up to the configured maximum limit (default: 15,000).  
    - `.voice 5000` ‚Äî Analyzes the last 5,000 messages.  
- `.voiceconfig [number_of_messages]` ‚Äî Sets the maximum message analysis limit.  
  - **Example:** `.voiceconfig 5000` ‚Äî Sets the limit to 5,000 messages.  
- `.vt` ‚Äî A shorthand for the `.voice` command.  

### üåü Features:  
- üéô Counts voice messages in the chat.  
- üìä Creates a leaderboard of active users.  
- ‚öôÔ∏è Customizable analysis message limit.
